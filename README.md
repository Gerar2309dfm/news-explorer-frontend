📰 News Explorer — Full Stack News Management Platform

Aplicación web Full Stack que permite a los usuarios buscar noticias en tiempo real, guardar artículos en una cuenta personal y administrar su colección de contenido favorito mediante una experiencia segura y personalizada.

El proyecto integra frontend y backend modernos utilizando React, Node.js, Express y MongoDB, implementando autenticación JWT, rutas protegidas y despliegue completo en producción.

📌 Funcionalidades principales
Registro de usuarios
Inicio y cierre de sesión
Autenticación segura mediante JWT
Persistencia de sesión
Búsqueda de noticias mediante API externa
Guardado de artículos favoritos
Eliminación de artículos guardados
Gestión personalizada de contenido
Rutas protegidas para usuarios autenticados
Comunicación Frontend ↔ Backend mediante API REST

🎯 Objetivo del proyecto

El objetivo fue desarrollar una plataforma que permitiera a los usuarios descubrir y gestionar noticias de interés mediante una experiencia personalizada y segura.

Este proyecto simula escenarios comunes de aplicaciones reales que requieren autenticación, gestión de contenido y comunicación entre frontend y backend.
